# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added

- Landing screen with Mom/Dad story selection
- Parent profile creation with name, birth year, city, photo
- 12 journey categories with 300 guided questions
- Voice recording with speech-to-text
- Text input with auto-save
- Question navigation with progress tracking
- Story timeline visualization
- AI chapter generation using OpenAI
- PDF book generation with premium typography
- Cover page, table of contents, chapters, and closing letter
- PDF export and sharing
- Firebase Firestore integration
- Firebase Analytics tracking
- Responsive design for mobile devices
- Premium editorial design system

### Technical

- Flutter 3.24+ compatibility
- Firebase Core, Auth, Firestore, Storage, Analytics
- OpenAI API integration
- Speech-to-text with Google Speech Recognition
- PDF generation with `pdf` and `printing` packages
- Image picker for photo uploads
- Share Plus for PDF sharing
