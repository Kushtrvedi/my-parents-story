# Product Constitution v1.0

> **Every family deserves to preserve the voice, wisdom, memories, and legacy of their parents — for free, forever.**

This document governs every product decision. No feature ships unless it aligns with these principles.

---

## The Five Non-Negotiable Pillars

### 1. Free Forever

The core experience will always remain free.

Includes:
- Unlimited parents
- Unlimited memories
- Unlimited voice recordings
- Unlimited chapters
- Unlimited PDFs
- Unlimited local storage
- Unlimited languages supported by the device

No ads. No subscriptions. No paywall around memories.

### 2. Privacy First

Parents' memories belong only to the family.

Default behavior:
- Store everything locally
- No mandatory account
- No cloud processing by default
- No selling or analyzing personal data
- Cloud backup is optional and user-controlled

### 3. Voice First

The app is designed for speaking, not typing.

Priority order:
1. Listen
2. Speak
3. Read
4. Type

Typing is a fallback.

### 4. Elderly First

Every design decision must answer:

> "Could my 75-year-old parent use this without help?"

Design standards:
- Large touch targets (minimum 60px)
- Large fonts (minimum 22px)
- High contrast
- Minimal text
- One primary action per screen
- No hidden gestures
- No technical terminology

### 5. Legacy Over Productivity

Never use:
- Tasks, goals, streaks, scores, XP, levels, achievement badges

Instead use:
- Chapters, memories, stories, legacy, family archive

---

## Product Principles

The app should never feel like:
- ChatGPT
- A journal
- A note-taking app
- A voice recorder
- A survey
- A chatbot

It should feel like:

> A patient grandchild sitting beside a parent, listening carefully and preserving every precious story.

## Complete Product Roadmap

### Phase 1 — Memory Capture (Current MVP)
- Parent profiles
- 300 guided questions
- Voice recording
- Transcription
- Photos
- Offline storage
- PDF memoir
- Family Archive

### Phase 2 — Memory Intelligence
- AI follow-up questions
- Related memory suggestions
- Automatic timelines
- Life event linking
- Chapter summaries

### Phase 3 — Family Legacy
- Family tree
- Search across memories
- Quotes collection
- Wisdom collection
- Photo albums
- Voice archive
- Interactive timeline

### Phase 4 — Future Generations
A child or grandchild can ask:
- "What was Grandpa's first job?"
- "What advice did Grandma give about marriage?"
- "What was Dad's happiest memory?"

Answers generated only from the parent's recorded stories, preserving authenticity.
