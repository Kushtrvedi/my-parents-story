# My Parents' Story

> Every Parent Carries A Library Inside Them.

An emotional legacy-preservation platform that transforms parents' life stories into beautifully formatted digital memory books through guided conversations.

![License](https://img.shields.io/badge/license-MIT-green)
![Flutter](https://img.shields.io/badge/Flutter-3.24+-blue)
![Dart](https://img.shields.io/badge/Dart-3.24+-blue)

## Overview

My Parents' Story helps families preserve their most precious memories before they're lost forever. Through guided questions and voice recording, users can capture their parents' life stories and transform them into beautiful, print-quality memoir books.

**This is not a productivity app, journaling app, or AI app. This is an emotional legacy-preservation platform.**

## Features

- **Guided Conversations** - 300 thoughtfully crafted questions across 12 life categories
- **Voice Recording** - Hands-free memory capture with speech-to-text
- **AI-Powered Writing** - Transforms fragmented answers into cohesive memoir chapters
- **Premium PDF Export** - Print-quality books with elegant typography
- **Beautiful Design** - Warm, editorial design that honors the content
- **Simple Experience** - No complexity, just memory preservation

## Installation

### Prerequisites

- Flutter 3.24+ ([Install](https://docs.flutter.dev/get-started/install))
- Firebase CLI (`npm install -g firebase-tools`)
- OpenAI API key ([Get one](https://platform.openai.com/api-keys))

### Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/my-parents-story.git

# Navigate to project
cd my-parents-story

# Install dependencies
flutter pub get

# Copy environment template
cp .env.example .env

# Configure Firebase (requires Firebase project)
flutterfire configure

# Add your OpenAI API key to .env
# OPENAI_API_KEY=sk-your-key-here

# Run the app
flutter run
```

### Detailed Setup

See [docs/Deployment.md](docs/Deployment.md) for complete setup instructions.

## Environment Variables

Create a `.env` file in the project root:

```env
# OpenAI API Key for AI chapter generation
OPENAI_API_KEY=sk-your-key-here

# Firebase (auto-generated by flutterfire configure)
# These are optional if using flutterfire CLI
FIREBASE_PROJECT_ID=
FIREBASE_STORAGE_BUCKET=
FIREBASE_API_KEY=
FIREBASE_APP_ID=
FIREBASE_MESSAGING_SENDER_ID=
FIREBASE_AUTH_DOMAIN=
```

**Never commit your `.env` file to version control.**

## Firebase Setup

1. Create a project at [Firebase Console](https://console.firebase.google.com)
2. Enable these services:
   - Authentication (Anonymous sign-in)
   - Cloud Firestore
   - Cloud Storage
   - Analytics
3. Run `flutterfire configure`
4. Deploy security rules from `docs/Security.md`

## AI Features

### Chapter Generation

When a user completes a category, AI transforms their answers into a cohesive memoir chapter:

- Preserves factual details
- Maintains authentic voice
- Creates flowing narrative
- Book-quality output

### Final Letter

AI generates a heartfelt closing letter based on all responses.

Generate print-quality memoir books including:
